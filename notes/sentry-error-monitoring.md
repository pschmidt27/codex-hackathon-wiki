# Sentry error monitoring

Sentry is a developer platform for error monitoring, debugging, and distributed tracing.

A German Sentry landing page describes it as "Fehlerüberwachung" and includes the copyright notice "© 2026 • Sentry ist eine eingetragene Marke der Functional Software, Inc.".

Sentry's docs page says it provides "end-to-end distributed tracing" to help developers identify and debug performance issues and errors across their systems and services.
