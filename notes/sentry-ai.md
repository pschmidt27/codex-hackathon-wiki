# Sentry for AI

Sentry’s AI docs cover two main use cases: tooling for AI assistants and AI features inside Sentry itself. For assistant workflows, it offers agent skills, an MCP server, a natural-language CLI, and monitoring for AI agents and MCP servers. For product features, it includes Seer Autofix, AI code review, replay summaries, and an AI-powered query builder.

The AI monitoring landing page frames the monitoring side as tracking LLM calls, tokens, and agent performance. It sits alongside the broader AI docs and reinforces that Sentry treats AI observability as part of its general monitoring platform.
