# Sentry for AI

Sentry’s AI docs cover two main use cases: tooling for AI assistants and AI features inside Sentry itself. For assistant workflows, it offers agent skills, an MCP server, a natural-language CLI, dotagents, and monitoring for AI agents and MCP servers. For product features, it includes Seer Autofix, AI code review, replay summaries, and an AI-powered query builder.

The AI monitoring landing page frames the monitoring side as tracking LLM calls, token usage, and tool executions. The top-level AI page also shows how the AI docs are organized into reusable building blocks for agents and product features.
