# Sentry documentation

Sentry is a developer-focused application monitoring platform with docs covering error tracking, performance monitoring, session replay, profiling, cron monitoring, logs, AI-related tools, and practical guides for instrumentation and alerting. The guides section helps users move from setup to deciding what to instrument, what to query, and what alerts to create, with topic pages for issues/errors, logs, metrics, traces, custom spans, and session replay.
