# Pi coding agent

Pi is a minimal terminal coding harness for adapting AI-assisted coding to a user’s workflow. It emphasizes extensibility through TypeScript extensions, skills, prompt templates, themes, and shareable Pi packages.

It supports multiple operating modes, including interactive use, print/JSON output, RPC, and SDK integration. The site also highlights broad model/provider support, tree-structured sessions with branching history, and export/share options.
