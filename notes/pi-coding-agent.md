# Pi coding agent

Pi is a terminal coding harness for AI-assisted development. It focuses on being extensible through TypeScript extensions, skills, prompt templates, themes, and reusable Pi packages.

It supports multiple ways to use the same toolchain, including interactive use, print/JSON output, RPC, and SDK integration. The project also emphasizes broad model/provider support, branching session history, and export/share options.
