# Sentry concepts & reference

Sentry’s concepts section collects the basic building blocks for working with the platform: key terms, search syntax and searchable properties, data management settings, migration to SaaS, and OpenTelemetry Protocol (OTLP) ingestion for traces and logs. It functions as a reference hub for the lower-level concepts behind Sentry’s observability workflows, especially search and data handling.
