# Sentry agent skills

Sentry’s official agent skills help AI coding assistants with two broad tasks: setting up Sentry in specific stacks and handling Sentry-related workflows such as debugging issues or reviewing PRs. The docs describe both a simple install flow and a declarative `dotagents` setup that records pinned versions in `agents.toml`.

Setup skills cover React, React Native, Python, Ruby, iOS, Next.js, .NET, Go, Svelte, AI monitoring, and OpenTelemetry exporter setup. Workflow skills cover fixing issues, PR code review, and alert creation. The docs also recommend keeping individual skills concise: prefer tables for reference material, use phased steps for multi-stage workflows, include version requirements where relevant, add troubleshooting tables, and target roughly 100–200 lines per skill.
